####### YOU ARE TRASH! #######

Jogo desenvolvido no projeto Gamerama usando a engine GTGE (http://www.goldenstudios.or.id/products/GTGE/index.php).

Requisitos mínimos:
Processador classe Pentium 4 1.0 GHz
Sistema operacional - Windows XP, Linux, MacOSX
Java 1.6 (JRE 1.6), disponível em http://www.java.com/en/download/inc/windows_upgrade_xpi.jsp

####### EXECUTANDO #######

ATENÇÃO!!!
Este software foi desenvolvido para execução em sistema operacional Windows XP 32-bit. Embora seja possível executá-lo em outros sistemas, não é garantida a funcionalidade completa, pois alguns podem apresentar problemas. Em nossos testes, foi detectado que alguns sistemas operacionais podem apresentar problemas de áudio, causando o travamento do jogo. Caso isso ocorra, recomenda-se desligá-lo na tela inicial pressionando as teclas "S" e "M".

WINDOWS:
Certifique-se de que todos os itens do pacote (ver seção CONTEÚDO) foram extraídos para o mesmo diretório. Execute o arquivo jogar_WIN.bat ou dê um duplo clique no arquivo LixoGame.jar

OUTROS SISTEMAS:
Abra um terminal, navegue para o diretório onde se encontra o arquivo LixoGame.jar e execute o comando (sem as aspas) "java -jar LixoGame.jar".

####### HISTÓRIA #######

O mundo passa por uma era de consciência ecológica após um quase apocalipse devido à poluição, onde reaproveitar e reciclar os materiais é vital para a sobrevivência do ser humano no planeta Terra. Porém, devido ao alto teor de poluentes, uma casca de banana que ia ser reciclada como adubo desenvolveu consciência e habilidades que somente um personagem de videogame teria.

Como Bananixo, seu objetivo é escapar da Central de Reciclagem dos humanos e alcançar a liberdade. Para isso, você deve atravessar os diversos estágios da central, usando os controladores de reciclagem para posicionar o fundo dos blocos de lixo com as cestas corretas. Passando desse estágio, Bananixo poderá atravessar a central enquanto o lixo é processado, mas cuidado!!! Humanos cometem erros e as cestas de reciclagem com lixo incorreto geram desafios impressionantes para Bananixo, como paredes de vidro ou lamaçal de lixo orgânico!

Vá Bananixo! O tempo não pára! Fuja da central de reciclagem antes que fique preso e transforme-se em adubo!

###### COMO JOGAR ######

PuzzleMode:
Desloque os blocos de lixo reciclável e rotacione-os de modo que o fundo seja da mesma cor da cesta de reciclagem sobre a qual ele cairá.

PlatformMode:
Leve Bananixo até o outro lado da tela, coletando os itens e destruindo ou fugindo dos desafios da fase, antes que o tempo acabe.


####### COMANDOS #######

NAVEGANDO PELO MENU

O menu consiste em 4 itens: Play, How To, Credits e High Score.

Setas para cima e para baixo - selecionam a opção no menu
Enter - acessa a opção selecionada
Esc - sai do jogo
s - habilita/desabilita som
m - habilita/desabilita música

PLAY

PuzzleMode

Seta para baixo - rotaciona o bloco no sentido horário
Seta para esquerda - rotaciona o bloco no sentido anti-horário
Seta para direita - rotaciona o bloco no sentido horário
Esc -  retorna ao menu

PlatformMode - Teste do platform mode

Seta para esquerda - faz o personagem andar para a esquerda
Seta para direita - faz o personagem anadar para a direita
Barra de espaço - faz o personagem pular
CTRL - em posse de uma pistola e com munição, atira
P ou pause - pausa/despausa o jogo
Esc - retorna ao menu

HOW TO - Como jogar

Setas para esquerda e direita - navegam entre as telas do HowTo
Enter - navega entre as telas do HowTo em sentido direto. Na última tela, retorna ao menu.
Esc - retorna ao menu

CREDITS - Créditos do jogo

Esc - retorna ao menu

HIGH SCORE - Exibição de recordes e pontuações mais altas

Esc - retorna ao menu
